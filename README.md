### 1.프로젝트 설명

#### 1-1. 프로젝트 기획
> 우리 사회 주변의 도움 및 대화를 필요로하는 사람들을 위한 대화 및 커뮤니티 앱 

#### 1-2. 프로젝트 목적
* AAC의 이해 및 구현
* MVVM 구조 이해를 위한 구현
* 메신저 및 실시간 채팅 서비스에 대한 이해
* http 통신의 이해

### 2. DoWoom 기능

* 실시간 채팅
* 커뮤니티
* 사진 전송
